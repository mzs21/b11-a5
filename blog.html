Question-1: What are the different ways to select an element in the DOM?

We can do in the following ways -
document.getElementById('elementId')
document.getElementsByClassName('className')
document.getElementsByTagName('tagName')
document.querySelector('.className');
document.querySelector('#elementId');
document.querySelectorAll('.className');

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets the HTML content inside an element.
innerText gets or sets the text content of an element, but does not get or set hidden elements.
textContent gets or sets the text content of an element. It gets the including hidden elements.

Question-3: What is event delegation in the DOM?

It's a technique. Event delegation involves using a single event listener for multiple elements. This listener will still work even if new elements are added.

Question-4: What is event bubbling in the DOM?

Event bubbling is a technique by which an event propagates from the target element up through its parent elements

Question-5: How do you create, add, and remove elements using JavaScript?

Create: document.createElement('tagName')
Add: parentElement.appendChild(newElement)
Remove: parentElement.removeChild(childElement)
